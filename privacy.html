Politique de confidentialite de Bricks Check

Derniere mise a jour : 18 mai 2026

Bricks Check est une extension Chrome qui surveille la page app.bricks.co ouverte par l'utilisateur afin de detecter les projets en collecte correspondant au seuil configure.

Bricks Check est une extension independante et non officielle. Elle n'est pas affiliee, associee, autorisee, approuvee ni sponsorisee par Bricks.co. Elle ne constitue pas un conseil en investissement.

Donnees traitees

L'extension recupere les donnees de projets directement depuis l'API publique Bricks (api.bricks.co) en utilisant le jeton d'authentification de la session Chrome de l'utilisateur. Ce jeton est lu uniquement lorsque app.bricks.co est ouverte dans le navigateur, et uniquement depuis le stockage local du site (localStorage / sessionStorage), avec une validation de format JWT.

Utilisation des donnees

Les donnees recuperees (noms de projets, briques disponibles, briques possedees) sont utilisees uniquement pour comparer les projets avec le seuil defini par l'utilisateur et afficher une notification Chrome lorsqu'un projet correspond aux reglages.

Stockage

Les reglages de l'utilisateur (seuil, intervalle de verification, options de notification) sont stockes via chrome.storage.sync. Le jeton d'authentification Bricks et le cache de briques possedees sont stockes localement via chrome.storage.local. Ces stockages ne quittent jamais l'appareil et ne sont pas chiffres : seules les autres extensions ayant la permission storage peuvent theoriquement y acceder.

Partage

Bricks Check ne vend, ne transfere et ne partage aucune donnee utilisateur avec des tiers.

Code distant

L'extension n'execute pas de code distant. Tout le code est inclus dans le package de l'extension.
